# MirrorDancer - 設計書

## 1. アプリ概要

**アプリ名:** MirrorDancer

ダンスの練習を効率化するモバイルアプリ。見本動画と自分の撮影動画を並べて比較再生し、練習の振り返りを支援する。

**対象プラットフォーム:** iOS / Android (Flutter)

---

## 2. 画面構成・遷移図

```
┌───────────────────────────────────────────────────┐
│                タブバー（常時表示）                   │
│  [⏯ 比較]  [🎬 見本動画]  [📹 自分の動画]  [⚙ 設定] │
└───┬──────────┬──────────────┬──────────────┬───────┘
    │          │              │              │
    ▼          ▼              ▼              ▼
  比較タブ   見本動画一覧  自分の動画一覧   設定
  (2モード)    │              │            ├─ 非表示の自分の動画
    │          │              │            └─ 非表示の見本動画
    │          │              │
    │          └──タップ──→ 比較タブ（比較モード）
    │                         │
    │                         └──タップ──→ 比較タブ（単体再生モード）
    │                              比較→ ボタンでも比較モードへ
    │
    ├─ 比較モード: 見本と自分の動画を左右並列再生
    └─ 単体再生モード: 自分の動画を単独再生
        （ヘッダー右上のトグルボタンでモード切替）
```

**遷移パターン:**
- タブ切替: タブバーからの直接遷移（タブバーは常時表示）
- 見本動画カードタップ → 比較タブ（比較モード）に遷移、該当見本を選択
- 自分の動画行タップ → 比較タブ（単体再生モード）に遷移、該当動画を選択
- 自分の動画行「比較→」ボタン → 比較タブ（比較モード）に遷移

## 3. データ構造

### 3.1 Reference（見本動画）

| フィールド | 型 | 説明 |
|---|---|---|
| id | String | 一意識別子 |
| title | String | タイトル（必須） |
| memo | String | 練習メモ（任意） |
| mirror | bool | ミラー反転フラグ |
| hidden | bool | 非表示フラグ |
| videoPath | String | 動画ファイルパス |
| createdAt | String | 作成日（YYYY-MM-DD） |

### 3.2 MyVideo（自分の動画）

| フィールド | 型 | 説明 |
|---|---|---|
| id | String | 一意識別子 |
| label | String | ラベル（自動生成 or ユーザー入力） |
| refId | String | 紐づく見本動画のID |
| date | String | 撮影日（YYYY-MM-DD） |
| hidden | bool | 非表示フラグ |
| videoPath | String | 動画ファイルパス |

### 3.3 データ連動ルール

- 見本動画を**非表示** → 紐づく自分の動画もすべて非表示
- 見本動画を**削除** → 紐づく自分の動画もすべて削除
- 見本動画を**再表示** → 紐づく自分の動画もすべて再表示
- 自分の動画の非表示/削除は単体で独立動作

## 4. 画面仕様

### 4.1 比較タブ（2モード統合）

**ヘッダー:** モード名（「比較」or「自分の動画」）+ モード切替ボタン
- 「📹 単体再生」/ 「⏯ 比較モード」トグル

#### 4.1.1 比較モード

**見本動画セレクター:**
- ドロップダウンで見本動画を切替（非表示の見本は除外）
- 選択中の見本タイトルと紐づく自分の動画本数を表示

**動画パネル（左右並列）:**
- 左: 見本動画（9:16縦長）、ミラー反転バッジ表示
- 右: 自分の動画（9:16縦長）、下部のサムネイルリストから選択

**個別再生コントロール（各動画下）:**
- ▶/⏸ 個別再生ボタン
- 速度バッジ（0.25x → 0.5x → 0.75x → 1x サイクル）
- 見本=青系（#5b9bd5）、自分=緑系（#6bcf7f）で色分け

**同時再生コントロール（中央）:**
- ▶▶ 同時再生/停止ボタン
- ↩ 5秒戻る / ↪ 5秒進む

**タイムライン:** シークバー + 経過時間/総時間

**開始位置合わせパネル（折りたたみ）:**
- ±0.1s / ±0.5s ボタンで自分の動画のオフセット調整
- ＋=自分が遅れてスタート / −=自分が先にスタート
- リセットボタン

**メモ表示:** 見本動画のメモを読み取り専用で表示

**自分の動画サムネイルリスト:**
- 横スクロール、先頭に「＋ 撮影/選択」ボタン
- 選択中は緑ボーダーでハイライト

#### 4.1.2 単体再生モード

**プレーヤー:** 16:9横長、動画名・日付・紐づく見本名を表示

**再生コントロール:**
- 速度バッジ（0.25x〜1x）
- ↩ 5秒戻る / ▶再生 / ↪ 5秒進む / 🔁リピート
- 緑系（#6bcf7f）のアクセントカラー

**動画リスト:**
- 🎬見本別 / 📅日付 トグル切替
- 折りたたみ対応、再生中動画のグループをハイライト
- 各グループ先頭に「＋ 撮影/選択」ボタン
- サムネイルタップで動画切替

#### 4.1.3 共通（両モード）
- 動画追加モーダル（カメラで撮影 / フォルダから選択）
- モード切替時、選択中の自分の動画IDは維持される

### 4.2 見本動画タブ

**ヘッダー:** 「見本動画」タイトル + ＋追加ボタン

**リスト:** 非表示でない見本動画をカード形式で表示
- サムネイル、タイトル、自分の動画本数バッジ、作成日
- メモ表示
- **カード全体タップ → 比較タブ（比較モード）へ遷移**
- アクション（カード下部、タップ遷移と干渉しない）: メモ編集 / ミラー反転ON/OFF

**スワイプ操作（左→右）:** オレンジ「非表示」+ レッド「削除」

**見本追加モーダル（ボトムシート）:**
- 動画ソース: 撮影 or フォルダから選択
- タイトル（必須）
- メモ（任意）

### 4.3 自分の動画タブ

**ヘッダー:** 「自分の動画」タイトル + 本数

**表示切替トグル:** 📅日付 / 🎬見本別

**グループ表示:**
- 日付ビュー: 日付ごとにグループ化（今日/昨日/MM/DD表記）
- 見本別ビュー: 見本動画ごとにグループ化
- 各グループは▼タップで折りたたみ可能

**各動画行:**
- **行全体タップ → 比較タブ（単体再生モード）へ遷移**
- サムネイル + ラベル + サブ情報（日付ビュー→見本名、見本ビュー→日付）
- 「比較 →」ボタン → 比較タブ（比較モード）へ（タップ遷移と干渉しない）

**スワイプ操作（左→右）:** オレンジ「非表示」+ レッド「削除」

### 4.4 設定タブ

**タブバー内の⚙ボタンから遷移（タブバーは常時表示）**

**メニュー:**
- 📹 非表示の自分の動画（件数表示）
- 🎬 非表示の見本動画（件数表示）

**非表示の自分の動画 サブ画面:**
- 各動画に「再表示」ボタン
- スワイプ（左→右）で削除のみ

**非表示の見本動画 サブ画面:**
- 各動画に「再表示」ボタン（紐づく自分の動画も再表示）
- スワイプ（左→右）で削除のみ（紐づく自分の動画も削除）

## 5. 共通UI仕様

### 5.1 スワイプ操作

| 対象 | 方向 | アクション |
|---|---|---|
| 見本動画カード | 左→右 | 非表示（オレンジ）+ 削除（レッド） |
| 自分の動画行 | 左→右 | 非表示（オレンジ）+ 削除（レッド） |
| 非表示リスト行 | 左→右 | 削除のみ（レッド） |

- スワイプ幅: 通常140px（2ボタン）、削除のみ80px
- 閾値: ACTION_WIDTH / 3 超えで開閉
- タッチ＆マウス両対応

### 5.2 動画追加フロー

全ての動画追加ポイントで共通のボトムシートモーダルを使用:
- 📷 カメラで撮影
- 📁 フォルダから選択

追加可能な箇所:
- 比較タブ（比較モード）のサムネイルリスト先頭
- 比較タブ（単体再生モード）の各グループ先頭
- 見本動画タブの「＋ 追加」ボタン（見本の追加）

### 5.3 カラースキーム

| 用途 | カラー |
|---|---|
| 背景 | #0f0f0f |
| サーフェス | #1a1a1a |
| ボーダー | #2a2a2a |
| テキスト | #e8e8e8 |
| テキスト（サブ） | #777 |
| アクセント | #4ecdc4 |
| 見本系 | #5b9bd5 / 背景 #0c1929 |
| 自分系 | #6bcf7f / 背景 #0c2919 |
| 危険（削除） | #ff6b6b |
| 警告（非表示） | #f0a050 |

### 5.4 速度変更

- 選択肢: 0.25x / 0.5x / 0.75x / 1x
- タップでサイクル切替
- 比較モード: 見本・自分それぞれ独立制御
- 単体再生モード: 1つの速度コントロール

## 6. 技術要件

### 6.1 Flutter パッケージ候補

| 機能 | パッケージ |
|---|---|
| 動画再生 | video_player / chewie |
| カメラ撮影 | camera / image_picker |
| ファイル選択 | file_picker / image_picker |
| ローカル保存 | sqflite / hive |
| 状態管理 | riverpod / provider |

### 6.2 動画同期再生の実装方針

- 2つのVideoPlayerControllerを独立管理
- 同時再生時は両方のplay()を同時呼び出し
- 開始位置オフセットはseekTo()で調整
- 個別速度はsetPlaybackSpeed()で制御

### 6.3 データ永続化

- ローカルDB（Hive or SQLite）に動画メタデータを保存
- 動画ファイル本体はアプリのドキュメントディレクトリに保存
- 見本動画のパスはフォルダ選択時のファイルパスを保持
