# 比較画面（CompareTab）設計書

## 概要

アプリの中核画面。見本動画と自分の動画を並べて比較再生する。
ファイル: `lib/screens/compare_tab.dart`（約 1690 行）

## モード

### 1. 比較モード (CompareMode.comparison)

見本動画と自分の動画を並べて同時再生する。

### 2. 単体モード (CompareMode.single)

自分の動画を単独で再生する。

モード切替は画面上部のセグメントコントロールで行う。

## 比較モードの構成

### レイアウト

```
┌─────────────────────────────┐
│ ナビゲーションバー            │
│ [見本動画セレクター ▼]        │
├──────────┬──────────────────┤
│          │                  │
│ 見本動画  │  自分の動画       │  ← ポートレート: 横並び (Row)
│          │                  │
├──────────┴──────────────────┤
│ [自分の動画サムネイル一覧 →]  │
├─────────────────────────────┤
│ ▓▓▓▓▓░░░░░  見本トラック     │  ← デュアルトラック タイムライン
│ ░░▓▓▓▓░░░░  自分トラック     │
├─────────────────────────────┤
│ [0.5x] [⏮] [◀5] [▶/⏸] [5▶]│  ← 統一コントロール
└─────────────────────────────┘
```

横長動画の場合は縦積み (Column) に自動切替。

### 動画パネル

各動画パネルの構成:
- `FittedBox` + `BoxFit.contain` でアスペクト比維持
- 見本動画: mirror フラグが true なら `Transform.flip(flipX: true)` で左右反転
- ラベル表示（左上: 「見本」青 / 「自分」緑）
- トリムアイコン（右上、タップで表示/非表示トグル）
- 空状態: タップで動画選択可能

### デュアルトラック タイムライン（C案: タイムライン型）

`CustomPainter` (`_TimelinePainter`) で描画:

```
時間軸（横方向）
├── 見本トラック（青 #5B9BD5）
│   ├── 背景バー
│   ├── 再生済み部分（塗りつぶし）
│   └── トリム範囲ハイライト
├── 自分トラック（緑 #6BCF7F）
│   ├── 背景バー（オフセット分ずらして表示）
│   ├── 再生済み部分（塗りつぶし）
│   └── トリム範囲ハイライト
├── 再生ヘッドライン（白い縦線 + ドット）
├── 時間ラベル（開始・中間・終了）
└── オフセットドラッグヒント矢印（◀ ▶）
```

**タイムライン操作:**
- タップ: その時間位置にシーク
- 自分トラック上でパン: オフセット調整
- それ以外でパン: スクラブ

### オフセット機能

`_offsetSeconds` (double) で見本と自分の動画の再生開始タイミングをずらす:
- 正の値: 自分の動画が遅れてスタート
- 負の値: 自分の動画が先にスタート
- タイムラインの自分トラックをドラッグして調整

### 統一コントロール

| ボタン | 機能 |
|-------|------|
| 速度ピル | 0.25x → 0.5x → 0.75x → 1.0x サイクル |
| ⏮ | 両動画を先頭に巻き戻し |
| ◀5 | 両動画を 5 秒戻す |
| ▶/⏸ | 同時再生/一時停止 |
| 5▶ | 両動画を 5 秒進める |

### トリミング機能

- 各動画パネル右上にハサミアイコン（1タップで表示/非表示）
- タップで CupertinoSlider ベースのトリムシート表示
- 開始時間・終了時間を設定可能
- 再生中にトリム範囲外に到達したら自動でトリム開始位置に戻す
- リセットボタンでトリム解除

## 比較モードの状態変数

```dart
// 動画コントローラー
VideoPlayerController? _refController;
VideoPlayerController? _myController;

// 再生制御
double _speed = 1.0;              // 再生速度
double _offsetSeconds = 0.0;      // オフセット（秒）
bool _syncPlaying = false;        // 同時再生中フラグ

// トリム
Duration _refTrimStart = Duration.zero;
Duration _refTrimEnd = Duration.zero;   // zero = 未設定（全体）
Duration _myTrimStart = Duration.zero;
Duration _myTrimEnd = Duration.zero;
bool _showRefTrimIcon = false;
bool _showMyTrimIcon = false;

// タイマー
Timer? _positionTimer;            // 再生位置更新用（16ms間隔）
bool _isDraggingOffset = false;   // オフセットドラッグ中
```

## 単体モードの構成

```
┌─────────────────────────────┐
│ ナビゲーションバー            │
├─────────────────────────────┤
│                             │
│    動画再生エリア              │
│                             │
├─────────────────────────────┤
│ [0.5x] [◀5] [▶/⏸] [5▶] [🔁]│
├─────────────────────────────┤
│ 見本A                       │
│   ├── 自分の動画1  [▶]       │
│   ├── 自分の動画2  [▶]       │
│ 見本B                       │
│   ├── 自分の動画3  [▶]       │
└─────────────────────────────┘
```

- 見本動画ごとにグループ化された動画リスト
- 各動画に個別再生ボタン
- ループ再生トグル

## 主要メソッド

| メソッド | 説明 |
|---------|------|
| `_syncPlay()` | 同時再生/一時停止トグル。再生時にタイマー開始 |
| `_rewindBoth()` | 両動画を先頭（トリム開始位置）に巻き戻し |
| `_seekToTime(double)` | タイムライン上の位置に両動画をシーク |
| `_cycleSpeed()` | 速度を次の段階に切替 |
| `_startPositionTimer()` | 16ms ごとに setState して再生位置を更新 |
| `_stopPositionTimer()` | タイマー停止 |
| `_onRefPlayback()` | 見本動画のトリム範囲強制リスナー |
| `_onMyPlayback()` | 自分動画のトリム範囲強制リスナー |

## 横長動画対応

`_isAnyLandscape` ゲッター:
- どちらかの動画のアスペクト比が 1.0 を超える場合 true
- true の場合、Row → Column に切替（縦積み）
- 各動画の高さ制約も自動調整
