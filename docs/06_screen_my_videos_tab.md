# 自分の動画画面（MyVideosTab）設計書

## 概要

ユーザーが録画した練習動画の一覧表示・管理を行う画面。
ファイル: `lib/screens/my_videos_tab.dart`（約 353 行）

## 画面構成

```
┌─────────────────────────────┐
│ 自分の動画 (N本)             │  ← CupertinoNavigationBar
├─────────────────────────────┤
│ [日付順 | 見本別]            │  ← グループ切替セグメント
├─────────────────────────────┤
│ ▼ グループ名 (3)            │  ← 折りたたみ可能ヘッダー
│ ┌─────────────────────────┐ │
│ │ ラベル名                 │ │
│ │ 2026-03-15 / 見本名      │ │
│ │                [比較→]  │ │
│ └─────────────────────────┘ │
│ ┌─────────────────────────┐ │
│ │ ...                     │ │
│ └─────────────────────────┘ │
└─────────────────────────────┘
```

## グループモード

| モード | グループキー | 表示 |
|-------|------------|------|
| 日付順 (byDate) | date | 「2026-03-15 (3)」 |
| 見本別 (byRef) | refId | 「見本動画名 (3)」 |

- `CupertinoSlidingSegmentedControl` で切替
- グループヘッダーはタップで折りたたみ/展開

## 動画行の構成

- ラベル名（16px）
- サブ情報: 日付順の場合は見本名、見本別の場合は日付
- 「比較→」ボタン: 紐づく見本を選択して比較タブへ遷移

### スワイプアクション（Slidable）

左スワイプ:
- 非表示（オレンジ）: `myVideosProvider.notifier.hide(id)`
- 削除（赤）: 確認ダイアログ → `myVideosProvider.notifier.delete(id)`

## 空状態

見本動画がない場合:
- 「まずは見本動画を追加してください」
- 「見本動画タブへ」ボタン → Tab 1 へ遷移

見本動画はあるが自分の動画がない場合:
- 「比較画面から自分の動画を追加してください」
- 「比較画面へ」ボタン → Tab 0 へ遷移

## ヘッダー

ナビゲーションバーの middle に動画数を表示:
```
自分の動画 (N本)
```
