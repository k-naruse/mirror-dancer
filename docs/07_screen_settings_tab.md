# 設定画面（SettingsTab）設計書

## 概要

非表示にしたアイテムの管理を行う画面。
ファイル: `lib/screens/settings_tab.dart`（約 328 行）

## 画面構成

```
┌─────────────────────────────┐
│ 設定                        │  ← CupertinoNavigationBar
├─────────────────────────────┤
│                             │
│ 非表示アイテム               │  ← セクションヘッダー
│ ┌─────────────────────────┐ │
│ │ 非表示の自分の動画  (N)  >│ │  ← メニュータイル
│ ├─────────────────────────┤ │
│ │ 非表示の見本動画    (N)  >│ │  ← メニュータイル
│ └─────────────────────────┘ │
│                             │
└─────────────────────────────┘
```

## 非表示アイテム管理画面

各メニュータイルをタップすると一覧画面に遷移（CupertinoPageRoute）。

### 非表示の自分の動画一覧

```
┌─────────────────────────────┐
│ ← 非表示の自分の動画          │
├─────────────────────────────┤
│ ┌─────────────────────────┐ │
│ │ ラベル名                 │ │
│ │ 日付                    │ │  ← Slidable
│ └─────────────────────────┘ │
└─────────────────────────────┘
```

スワイプアクション:
- 再表示（緑）: `unhide(id)`
- 完全削除（赤）: 確認ダイアログ → `delete(id)`

### 非表示の見本動画一覧

同様の構成。再表示時は紐づく MyVideo も連動して再表示。

## UI パターン

- メニュータイル: `Container` + `GestureDetector` でカスタムスタイル
- カウントバッジ: 非表示アイテム数を表示
- 空状態: 「非表示のアイテムはありません」
